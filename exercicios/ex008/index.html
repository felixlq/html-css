Principais formatações


Negrito / Destaque

Nesta frase temos um texto em Negrito usando a tag B (não semântica).

Nesta frase temos um termo em destaque usando a tag STRONG (semântico).


Itálico / Ênfase

Nesta frase, temos um termo em Itálico usando a tag I (não semântico)

Nesta frase, temos um termo em ênfase usando a tag EM (semântico)


Texto Marcado

Podemos criar também um texto marcado pela tag MARK

E no outro parágrafo temos um outro texto Marcado

Agora sempre a cor do MARK defult será dado pelo seletor.


Texto Grande e Pequeno

Criando um texto Grande com a tag BIG (obsoleta) e um texto Pequeno com a tag Small.


Texto Deletado

Podemos marcar um texto com excluído para indicar que ele deve ser lido mas não considerado. Com a TAG DEL


Texto Inserido

Podemos marcar um texto com incluído para indicar ênfase que ele foi adicionado depois. Com a TAG INS

Pode-se sublinhar também com a TAG U (Obsoleta e não semântica)


Texto Sobrescrito

Para inserir coisas do tipo x20+552 Com a TAG SUP


Texto Subscrito

Para inserir coisas do tipo C6H12O6 Com a TAG SUB
